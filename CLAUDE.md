## API documentation (api.json)

This repo commits a snapshot of the OpenAPI spec at `api.json`, generated by `dedoc/scramble` from route/controller attributes (`#[BodyParameter]`, `#[ApiResponse]`, etc. in `app/Http/Controllers/*`). It bakes in `APP_URL`/`APP_NAME`, which differ between local dev, CI, and production.

**Whenever you change a documented API endpoint (routes under `routes/api.php`, or their controller's Scramble attributes), regenerate `api.json` with:**

```sh
composer docs:export
```

**Do not** run `php artisan scramble:export` directly — it will bake in your local `.env`'s `APP_URL`/`APP_NAME` instead of production's, producing a diff that fails CI (`.github/workflows/ci.yml` re-runs `composer docs:export` and fails the build if `api.json` doesn't match).

Run `php artisan scramble:analyze` first if you want to check for documentation-generation errors without writing the file.
